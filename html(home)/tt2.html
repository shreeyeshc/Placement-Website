    Edit Distance
    Minimum Partition
    Ways to Cover a Distance
    Longest Path In Matrix

    Shortest Common Supersequence
    Matrix Chain Multiplication
    Partition problem

    Coin change problem
    Word Break Problem
    Maximal Product when Cutting Rope
